# Provider reference — Model providers

## When to use

Use this when the app will actually run agent responses.

## Default recommendation

Choose the model provider explicitly and record:

- provider
- model id
- API key owner

## Need from the user

- provider choice
- model choice
- owner of the credentials
- expected environment(s)

## Traps to avoid

- don't treat model choice as implied by deploy platform
- don't forget ownership of API keys in the handoff plan
