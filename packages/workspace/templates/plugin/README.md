# Workspace Plugin Template

Copy this folder into a workspace package or app plugin directory, then delete
the files you do not need. Replace `sample` ids before use.

Expected shape:

```txt
samplePlugin/
  front/
    index.tsx
    panels.tsx
    catalogs.ts
    surfaceResolver.ts
    bindings.tsx
    __tests__/samplePlugin.test.ts
  server/
    index.ts
  shared/
    constants.ts
    types.ts
```
